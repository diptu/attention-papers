# 2015 — The Expansion of Attention

## Theme

Attention evolved from a translation mechanism into a general computational
primitive for vision, memory, and summarization.

---

## Major Research Areas

- Attention Mechanisms
- Memory Networks

---

## Key Papers

1. Effective Approaches to Attention-based NMT
2. Show, Attend and Tell
3. Neural Attention Model for Abstractive Summarization 
4. Memory Networks
5.  End-to-End Memory Networks

---

## Major Contributions

- Generalized neural attention
- Introduced external differentiable memory
- Expanded attention beyond machine translation
- Multi-hop reasoning

---

## Influence

These ideas directly influenced:

- Key-Value Memory Networks (2016)
- Transformer (2017)
